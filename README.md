# CLI Todo Manager 📝

Простая утилита для управления списком дел прямо из терминала. Никаких лишних графических интерфейсов и тяжелых зависимостей — только чистый Python и JSON-файл для хранения данных.

Писал для себя, чтобы не забывать мелкие дела во время работы за компом.

## Как пользоваться

Все управление идет через команды в терминале:

* **Показать все задачи:**
  `python main.py list`

* **Добавить дело:**
  `python main.py add "Купить молоко"`

* **Отметить как выполненное:**
  `python main.py done 0` *(где 0 — номер задачи из списка)*

* **Удалить задачу:**
  `python main.py delete 0`

## Архитектура
Не люблю сваливать весь код в одну кучу, поэтому проект разбит на 4 понятных модуля:
- `main.py` — обрабатывает ввод команд (CLI).
- `tasks.py` — логика: добавить, удалить, изменить статус.
- `storage.py` — чтение и запись в JSON.
- `config.py` — настройки (пути к папкам).

## Установка
Ничего устанавливать не нужно, проект работает на стандартных библиотеках Python. Просто скачай код и запускай.

```bash
git clone https://github.com/shull1x/todo_app.git
cd todo_app
python main.py list
```
